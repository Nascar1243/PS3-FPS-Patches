These files work with Artemis to allow you to unlock the FPS in certain games, more games to come, there are also graphics settings that can be changed if you want to make games look sharper.

1. To install these new files, you first need to go to webman, then hit update webman, you aren't there to update webman though, you just want to scroll down to where it says Artemis GUI then install that.
2. Once it is installed open the application and wait for the cheats database to download.
3. Once it is finished you want to FTP into the console and copy the files you want to dev_hdd0/game/ARTPS3001/USDIR/USERLIST.
4. After this navigate to Options, then scroll down to plugin version and change it to webMAN MOD.
5. Back out of Options and go to Cheats.
6. Here you can filter by what games you have installed on the console, although sometimes they don't detect properly.
7. Scroll to the game you want and in the game cheats hit "Unlock FPS" or "60 FPS" and then whatever else you want.
8. Back out of the cheats and then hit Start Game.
9. Then load up the game you applied cheats for and have fun.
10. Make sure you are on the correct update version, most of the time it will apply to the latest update.
11. When prompted via notification, you will hit the home button and then start and the patch should apply.
12. If you don't see the notification then repeat steps 5-11 again and it should work.

Note: You will have to do this everytime you open the game.
